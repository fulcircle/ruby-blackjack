ruby-blackjack
==============

Simple example of console BlackJack game in ruby

To run:

<code>ruby blackjack.rb</code>


Requirements:

Ruby v1.9

House Rules:

* Dealer draws until soft 17 or higher
* Can either split or double-down on first turn ONLY (no double-after-split)
* If dealer and player both have blackjack or 21, this resuls in a push (player receives back the hand's bet amount)
* BlackJack pays 3:2, all other hand wins pay 1:1
* No insurance bets

Unit tests in 'test.rb' file
